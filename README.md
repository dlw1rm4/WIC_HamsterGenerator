## <img src="public/images/expressions/neutral.jpeg" alt="Hamster Vision" width="20"> Description

**Hamster Vision** is a one of a kind interactive web app that uses your webcam and real time emotion detection to match your face to the perfect hamster meme. Simply point your face at the camera, and the app analyzes your expression to find which hamster you are!

Built as a Winter 2026 WIC project, Hamster Vision uses MediaPipe's FaceLandmarker API to detect facial landmarks and classify emotions in real time, mapping your expression to a curated database of hamster memes. Whether you're happy, sad, or just existing, there's a hamster for that. The app also features a pixel art aesthetic, animated backgrounds, and a Day / Night / Pink theme switcher for extra personality.

**Key Features:**
- 🎥 Live webcam face detection
- 🐹 Matches YOUR face to a cute hamster meme!
- 🎨 Day / Night / Pink theme switcher
- ⚡ Runs entirely in the browser

## <img src="public/images/expressions/excited.jpeg" alt="Hamster Vision" width="20"> Explanation of Top-Level Files & Folders

| File / Folder | Description |
|---|---|
| `src/` | Main source code: TypeScript logic for face detection and meme matching |
| `public/` | Static assets including hamster images and pixel art backgrounds |
| `dist/` | Built/compiled output files generated by Vite |
| `.vercel/` | Vercel deployment configuration |
| `index.html` | Landing page with the animated sky background and Get Started button |
| `camera.html` | Camera page where face detection and hamster matching happens |
| `vite.config.js` | Vite build tool configuration |
| `vercel.json` | Vercel deployment routing settings |
| `tsconfig.json` | TypeScript compiler configuration |
| `package.json` | Project dependencies and scripts |
| `.gitignore` | Files excluded from version control |
